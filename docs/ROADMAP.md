# Roadmap

## Phase 0: Foundation (Month 0-2)

- Finalize architecture and threat model.
- Define kernel/service IPC contracts.
- Stand up CI, coding standards, and signing pipeline.

## Phase 1: Minimal Bootable System (Month 2-6)

- Boot chain + secure boot prototype.
- Kernel primitives: process, memory, IPC.
- Minimal shell, package manager stub, and update service skeleton.

## Phase 2: Security & UX Core (Month 6-10)

- App sandbox and permissions framework.
- Policy engine and credential service.
- Basic compositor/windowing and settings app.

## Phase 3: Compatibility & Developer Platform (Month 10-16)

- Compatibility runtime MVP.
- SDK/CLI, package format, developer docs.
- Performance profiling and hardware validation tiers.

## Phase 4: Public Alpha (Month 16+)

- Installer image and OTA updates.
- Desktop usability and stability pass.
- Security audit and bug bounty launch.
