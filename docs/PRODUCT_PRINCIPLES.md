# Product Principles

1. Security First
   - Least privilege everywhere.
   - Signed binaries and reproducible builds.
   - Sandboxing for all untrusted apps.

2. Performance First
   - Low memory footprint baseline.
   - Fast boot and responsive UI.
   - Resource-aware scheduling.

3. User Control
   - Fine-grained privacy settings.
   - Replaceable components where safe.
   - Scriptable system behavior.

4. Compatibility
   - Strong package/runtime story.
   - Compatibility layer for legacy apps.
   - Stable APIs and long-term support policy.

5. Polished Experience
   - Clear design language.
   - Consistent behavior across devices.
   - Minimal friction for everyday tasks.
